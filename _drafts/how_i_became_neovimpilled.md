## The start of my coding journey

<!-- markdownlint-disable MD013 -->

The first time I wrote code was during my first year of highschool, when I did a summer course in computer science for high school students at PUC-Rio (notably the university where Lua was created) with some friends.
We were supposed to make a game using a game engine the course provided in C, and we all learned how to use VS Code. At this point, besides VS Code, the only code editor I knew of where Sublime and Atom, because those where the ones I found on the internet recommended for Mac users.
A few months later, I started learning Python and Anaconda on my own. I tried Jupyter Notebook and Spider, and wrote a small project for a fair in my school.
